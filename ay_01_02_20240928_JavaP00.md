###### 练习 数组
```java
//数组排序
int[] arr = {12,14,23,45,66,68,70,77,90,91};
int[] brr = new int[11];
Scanner sc = new Scanner(System.in);
System.out.println("请输入：（数字）");
int num = sc.nextInt();
int index = 0;
for (int i = 0; i < arr.length; i++) {
    if(arr[i] <= num) {
        brr[i] = arr[i];
        index = i + 1;
    }else{
        brr[i+1] = arr[i];
    }
}
brr[index] = num;
for (int i = 0; i < brr.length; i++) {
    System.out.print(brr[i] + " ");
}
```
